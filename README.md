# Torneio Mythology MD3

## Como usar
1. Abra o site por `http://localhost` ou GitHub Pages.
2. Não use `file:///`.
3. Deixe a planilha do Google Sheets pública para leitura.
4. Se você importar a nova planilha como um arquivo novo no Google Sheets, troque o `sheetId` em `config.js`.

## Estrutura da planilha
- Config
- Equipes
- Oitavas
- Quartas
- Semifinal
- Triangular
- Final
- Patrocinadores
- Campeoes
- Deuses

## O que é automático
- Triangular puxa vencedores da Semifinal
- Final puxa vencedores da Semifinal 1 e 2
- Nomes das equipes e dos jogadores são puxados pelo TimeID
- Score da série é contado a partir de Jogo1/Jogo2/Jogo3

## O que você preenche
Somente células amarelas na planilha.
